# PPL Advanced Knowledge Base

Use this reference with `goal-hypertrophy.md` and `hypertrophy-splits.md` when the user asks about 三分化, PPL, 推力日, 拉力日, 腿日, push/pull/legs, 卧推技术, 背阔张力, 肩胛控制, 10+5 递进, 髋/后链/足踝, 每周 6 练 PPL, 4-5 练滚动三分化, 3 练保守版, 专项插入, or how ordinary trainees should adapt advanced PPL ideas.

## Positioning

Core definition: 三分化 is not just arranging chest, back, and legs casually. It uses PPL as the base skeleton to organize main-lift capability, target-muscle tension, training variables, recovery management, and long-term progression.

Compared with a simple PPL template, this reference adds bench technique, scapular control, lat tension, hip/posterior-chain/foot-ankle stability, RPE/RIR, 10+5 progression, boundaries for low-volume high-intensity training, and specialization-day insertion rules.

## User tiers

| User type | Programming stance |
|---|---|
| New trainee | Do not copy high-intensity templates. Build movement patterns, RPE awareness, and target-muscle control first. |
| Ordinary hypertrophy trainee | Use moderate volume, medium-high intensity, stable execution, and a PPL or rolling 三分化 structure. |
| Intermediate or advanced trainee | Insert specialization days by weak point and use 10+5, drop sets, density progression, and other advanced variables selectively. |
| Skills knowledge base | Treat this file as decision rules + movement logic + variable control + template generation for 三分化. |

## Decision flow

1. Identify the user's primary goal: hypertrophy, fat-loss muscle retention, strength, posture/body-shaping, or weak-point specialization.
2. Classify training level and execution control: new, ordinary, intermediate, advanced, or high-level.
3. Match frequency: 3, 4, 5, or 6 training days require different PPL execution.
4. Identify weak points: chest, shoulder, lat, mid-back, posterior chain, quads, arms, or general balance.
5. Give the PPL skeleton: push, pull, legs with main lifts and movement slots.
6. Add variables: sets, reps, RPE/RIR, rest, progression, and deload rules.

## PPL skeleton

| Day | Default muscles | Core capability | Common main movements |
|---|---|---|---|
| Push | Chest, front/middle delts, triceps | Bench stability, scapular platform, chest/shoulder/triceps linkage | Barbell bench press, incline press, shoulder press |
| Pull | Back, rear delts, biceps | Scapular depression, lat tension, row trajectory | Pull-up/pulldown, chest-supported row, single-arm pulldown |
| Legs | Quads, glutes, hamstrings, calves, core | Hip hinge, pelvic stability, foot/ankle support, unilateral control | Squat/hack squat, Romanian deadlift, Bulgarian split squat |

Rules:

- PPL is the base classification, not a fixed template.
- Add chest, shoulder, lat, posterior-chain, or arm specialization only when weak points and recovery justify it.
- Main compound lifts develop capability and screen technique; do not force every compound set to failure.
- Progress by target reps and execution quality before load. Isolation work progresses by control, reps, density, tension, peak contraction, drop sets, or small load jumps.

## Training variables

Load in kg is not universal intensity. The same 80 kg bench can be a warm-up, working set, or max-effort set for different users. Prescribe with RM, RPE/RIR, execution quality, and rest.

| Expression | Meaning | Best use |
|---|---|---|
| 80 kg x 8 | Weight and reps only | Logs, not enough for cross-user programming |
| 80 kg x 8 @RPE 8 | About 2 reps in reserve | Main-lift prescription |
| 8-12 reps, RIR 1-2 | Effort controlled by reps in reserve | Most hypertrophy movements |
| 10RM | Weight the user can do for about 10 max reps | Estimating training intensity |

| RPE | RIR | Meaning | Use |
|---|---|---|---|
| 6 | 4 | Easy, plenty in reserve | Warm-up, technique learning, recovery |
| 7 | 3 | Training stimulus but not hard | New exercises, early cycle |
| 8 | 2 | High-quality main-lift intensity | Bench, squat, rows, presses |
| 9 | 1 | Very close to failure | Limited peak sets or isolations |
| 10 | 0 | True failure | Mostly final isolation sets, used cautiously |

Default intensity: main compound lifts usually stay around RPE 7-8.5, with 1-3 reps in reserve. Isolation movements can move closer to failure, especially lateral raises, flyes, rear-delt flyes, curls, pressdowns, and leg extensions.

| Movement or intensity | Suggested rest | Quality standard |
|---|---|---|
| Heavy sets around 5 reps | 3-8 min | Next set keeps technique and speed |
| 6-8 rep compounds | 2-5 min | No form breakdown or abnormal RPE jump |
| 8-12 rep hypertrophy sets | 30 sec-3 min | Adjust by goal, complexity, and pump |
| 12-20 rep isolations | 45-120 sec | Target muscle, not cardio or compensation, limits the set |
| Density techniques | Shorter rests | Use on safe isolation or machine work only |

Frequency and weekly volume:

- Most major muscles usually do well with about 2 weekly exposures.
- Major muscles can start around 10-16 hard sets per week; increase only when recovery and execution remain stable.
- Smaller muscles often start around 6-12 hard sets per week; middle delts, rear delts, and arms can tolerate higher reps and frequency.
- Do not cram excessive work for one muscle into one session when quality drops; split it across another exposure.

## Push day

Push day = chest + front/middle delts + triceps. Bench variations develop push capability and screen upper-body stability, but push day is not a pure chest day.

| Slot | Recommended movements | Purpose |
|---|---|---|
| Main push | Barbell bench press, incline barbell press, dumbbell bench press | Overall push capability, chest/shoulder/triceps coordination |
| Assisted push | Incline dumbbell press, machine chest press, seated shoulder press | Add volume, change angle, reduce free-weight technical demand |
| Chest isolation | Cable fly, pec deck, dumbbell fly | Horizontal adduction, stretch and contraction tension |
| Middle delt | Dumbbell lateral raise, cable lateral raise, machine lateral raise | Shoulder width stimulus not fully covered by pressing |
| Triceps | Rope pressdown, overhead extension, close-grip press | Elbow extension and arm volume |

Bench checklist:

| Cue | Correct requirement | Common error |
|---|---|---|
| Unrack | Keep scapular platform and pull the bar out rather than shrugging it up | Scapula loses position during unrack |
| Scapula | Retract, depress, and slightly posteriorly tilt for a stable platform | Shoulders roll forward and chest tension disappears |
| Thoracic spine | Moderate arch with stable scapula | Forcing a fake arch through the low back |
| Legs | Feet stable, leg drive builds full-body tension | Feet move, hips leave bench, no lower-body involvement |
| Elbows | Avoid extreme tuck or flare; find a stable shoulder angle | Excessive flare stresses anterior shoulder |
| Descent | Controlled eccentric, stable path, consistent touch point | Bouncing off chest or wandering bar path |
| Press | Chest, triceps, and shoulder work together without shoulder dumping forward | Shoulders shoot forward and chest tension is lost |

Push-day main lift principle: bench-style main lifts usually stay around RPE 8 with 1-2 reps in reserve. The goal is stable progression and technique screening, not proving effort by failing reps.

Sample push day:

| Movement | Sets x reps | Intensity | Note |
|---|---|---|---|
| Barbell bench press | 3-4 x 5-8 | RPE 7.5-8.5 | Main lift, execution first |
| Incline dumbbell press | 3 x 8-12 | RIR 1-2 | Upper chest and push volume |
| Seated or machine shoulder press | 2-3 x 6-10 | RIR 1-2 | Do not max out when bench is heavy |
| Cable fly or pec deck | 2-3 x 12-20 | RIR 0-2 | Control stretch and contraction |
| Lateral raise | 3-5 x 12-25 | RIR 0-2 | Control first, no blind load jumps |
| Pressdown or overhead extension | 2-4 x 10-20 | RIR 0-2 | Can use peak contraction or drop sets |

Lateral raise, fly, and rear-delt fly rules:

- Do not progress them like bench, squat, or deadlift.
- Prioritize target-muscle tension, trajectory, eccentric control, peak contraction, and reduced compensation.
- Progress with reps, control, shorter rest, peak pauses, drop sets, or small valid load jumps.
- Wrong progression: shrugging, swinging, shortening range, or letting traps take over just to add load.

## Pull day

Pull day = back + rear delts + biceps. Back is not one generic muscle group; distinguish lats, upper/mid-back, lower traps, rear delts, and biceps.

| Region | Movements | Technical target |
|---|---|---|
| Lats | Pulldown, pull-up, single-arm cable pulldown, straight-arm pulldown | Scapular depression, elbow toward hip, sustained tension |
| Mid-back/thickness | Chest-supported row, seated row, barbell row | Scapular retraction control, elbow back |
| Rear delts | Reverse pec deck, rear-delt fly, rear-delt row | Avoid trap compensation, control peak |
| Biceps | Dumbbell curl, preacher curl, hammer curl | Stable elbow position, no body swing |

Pull-day priority: tension > trajectory > scapular control > load. Add weight only after the target muscle reliably takes the work.

Lat technique:

- Scapula first: depress the scapula before the pull; avoid shrugging the movement into upper traps.
- Elbow path: drive the elbow toward the hip, not just the hand toward the chest.
- Arm role: hands are hooks; reduce biceps takeover.
- Range: range of motion serves target-muscle tension, not the other way around.
- Eccentric: lengthen the lat without fully unloading it.

Single-arm cable pulldown execution:

| Phase | Standard | Error |
|---|---|---|
| Start | Lat lengthened, scapula controlled, no uncontrolled shrug | Shoulder pulled up near ear and tension lost |
| Pull | Scapula depresses first, elbow moves toward hip | Hand and biceps dominate |
| Contraction | Lat tight without turning the movement into a row | Torso leans back and elbow goes too far behind body |
| Return | Controlled eccentric, lat stays loaded | Fast rebound and full unloading |

10+5 progression:

10+5 means completing 10 standard reps, briefly pausing or slightly resetting, then adding 5 quality reps. Its value is effective reps and target-muscle stimulus, not direct load chasing.

| Stage | Execution |
|---|---|
| 1 | Ordinary sets are stable and the user can feel target muscle clearly. |
| 2 | Add 10+5 only on the final set. |
| 3 | Add 10+5 to more sets while preserving path and tension. |
| 4 | Add a small valid load increase only after execution remains stable. |

Sample pull day:

| Movement | Sets x reps | Intensity | Note |
|---|---|---|---|
| Pulldown or pull-up | 3-4 x 6-10 | RIR 1-2 | Lat main slot, scapular depression |
| Single-arm cable pulldown | 3 x 10-15 | RIR 1-2 | Lat path and unilateral control |
| Chest-supported row | 3 x 8-12 | RIR 1-2 | Mid-back thickness, less low-back compensation |
| Straight-arm pulldown | 2-3 x 12-20 | RIR 0-2 | Lat isolation, sustained tension |
| Rear-delt fly | 3 x 12-25 | RIR 0-2 | Control first |
| Curl or hammer curl | 2-3 x 8-15 | RIR 0-2 | Biceps supplement, not allowed to steal back work |

## Legs day

Legs day is lower body + hips + posterior chain + pelvic stability + foot/ankle function. It is not only a quad-blasting day.

| Module | Movements | Purpose |
|---|---|---|
| Knee-dominant | Squat, hack squat, leg press, leg extension | Quads and knee extension |
| Hip/posterior chain | Romanian deadlift, single-leg deadlift, back extension, hip thrust | Hamstrings, glutes, hip hinge |
| Unilateral stability | Bulgarian split squat, lunge, step-up | Pelvic stability, left-right balance, glute med involvement |
| Foot/ankle/calves | Standing calf raise, seated calf raise, tibialis work | Support capacity, ankle control, complete calf stimulus |
| Core stability | Anti-rotation, dead bug, hanging leg raise | Trunk fixation for lower-body force |

Dynamic preparation:

| Area | Purpose | Options |
|---|---|---|
| Foot/ankle | Support and proprioception | Ankle mobility, calf warm-up, tibialis activation |
| Hip | Mobility and hip control | Dynamic hip opener, hip flexion/extension drill, bodyweight hinge |
| Pelvis | Neutral position and anti-rotation | Dead bug, bird dog, single-leg balance |
| Posterior chain | Hamstring and glute tension | Glute bridge, light RDL, back extension warm-up |
| Main lift | Ramp toward working intensity | Empty bar or light ramp sets, not direct heavy loading |

Hinge progression before heavy deadlifts:

| Stage | Priority |
|---|---|
| 1 | Bodyweight hinge: hips fold back, spine stable, hamstrings lengthen. |
| 2 | Light Romanian deadlift: controlled eccentric, posterior-chain loading. |
| 3 | Back extension, hip thrust, single-leg deadlift: posterior chain and pelvic stability. |
| 4 | Conventional deadlift or heavier pulling only after the pattern is mature. |

Leg-day main rule: prioritize hips, posterior chain, pelvic stability, and unilateral capacity. Do not make conventional deadlift a blind heavy main lift for beginners or high-fatigue phases.

Sample legs day:

| Movement | Sets x reps | Intensity | Note |
|---|---|---|---|
| Hack squat or squat | 3-4 x 5-10 | RPE 7.5-8.5 | Quad main slot, stable execution |
| Romanian deadlift | 3 x 6-10 | RIR 1-2 | Hip hinge and posterior-chain tension |
| Bulgarian split squat | 2-3 x 8-12/side | RIR 1-2 | Unilateral stability and glute/leg control |
| Leg extension | 2-3 x 12-20 | RIR 0-2 | Quad isolation, can approach failure |
| Seated or prone leg curl | 2-3 x 10-15 | RIR 0-2 | Hamstring knee-flexion function |
| Calf raise | 3-5 x 8-20 | RIR 0-2 | Use straight-knee and bent-knee angles when useful |

## Periodization and progression

Progress is not only load increases.

| Progression type | Example | Best for |
|---|---|---|
| Add load | Bench 60 kg -> 62.5 kg | Stable main lifts and compounds |
| Add reps | 60 kg x 8 -> 60 kg x 10 | Almost all movements |
| Add sets | 3 sets -> 4 sets | When volume is low and recovery is good |
| Lower RPE | Same load/reps moves from RPE 9 to RPE 8 | Main-lift capability improvement |
| Improve execution | More stable path and less compensation | All movements, especially newer users |
| Improve control | Better eccentric and peak contraction | Raises, flyes, pulldowns, curls |
| Shorten rest | 120 sec -> 90 sec with same quality | Isolation and density work |
| Intensity technique | Final-set 10+5 or drop set | Isolation, machine, and safe movements |

Main-lift progression sequence:

1. Learn the pattern and force sequence.
2. Make working-set execution consistent.
3. Complete target reps, such as all 3 sets reaching 8-10 reps.
4. Add volume only if recovery allows, such as 3 sets to 4 sets.
5. Add load by the smallest valid jump.
6. Enter lower-volume higher-intensity phases only when execution remains stable.
7. Deload by reducing sets 30-50% and lowering load or RPE.
8. Start the next cycle by rebuilding volume and technique quality.

Specialization insertion:

| Weak point | Insertion method | Caution |
|---|---|---|
| Chest | Add chest specialization or make Push A chest-dominant | Avoid front delts/triceps fatiguing before chest work |
| Shoulder | Make Push B shoulder-dominant and increase lateral-raise frequency | Control before load on lateral raises |
| Lats | Make Pull A lat-dominant with single-arm pulldowns and straight-arm pulldowns | Reduce biceps and trap compensation |
| Mid-back | Make Pull B row/chest-supported-row dominant | Manage low-back fatigue; do not make every row free-weight heavy |
| Posterior chain | Make Legs B RDL, hip thrust, leg curl, and back-extension dominant | Hinge pattern first, deadlift load later |
| Arms | Add short arm specialization at push/pull endings or as a short session | Do not reduce main-lift performance |

## Templates

Six-day advanced PPL:

| Day | Theme | Focus |
|---|---|---|
| Day 1 | Push A: chest-dominant | Bench main lift, upper-chest supplement, lateral raise, triceps |
| Day 2 | Pull A: lat-dominant | Pulldown/pull-up, single-arm cable pulldown, straight-arm pulldown, rear delts |
| Day 3 | Legs A: quad-dominant | Squat/hack squat, leg press, leg extension, small posterior-chain dose |
| Day 4 | Rest or active recovery | Light cardio, mobility, sleep, nutrition recovery |
| Day 5 | Push B: shoulder/upper-chest dominant | Incline press, shoulder press, lateral raise, fly, triceps |
| Day 6 | Pull B: mid-back/rear-delt dominant | Chest-supported row, seated row, rear delts, biceps |
| Day 7 | Legs B: posterior-chain/glute dominant | RDL, hip thrust, Bulgarian split squat, leg curl, calves |

Four-to-five-day rolling 三分化:

| Order | Arrangement |
|---|---|
| 1 | Push |
| 2 | Pull |
| 3 | Rest |
| 4 | Legs |
| 5 | Push |
| 6 | Rest |
| 7 | Pull |
| 8 | Legs |
| 9 | Rest, then repeat |

Use this for natural trainees, average recovery, unstable schedules, or users who need each muscle stimulated roughly every 4-5 days without forcing six sessions.

Three-day conservative version:

| Weekly day | Content |
|---|---|
| Monday | Push: bench/incline press + lateral raise + triceps |
| Wednesday | Pull: pulldown/row + rear delts + biceps |
| Friday | Legs: squat/leg press + RDL/leg curl + calves + core |

Use this for low time availability, new trainees, poor recovery, or habit building. It is not the strongest hypertrophy setup, but it can be a good entry or transition plan.

## Ordinary vs high-level trainees

| Dimension | High-level trainee | Ordinary trainee/new trainee |
|---|---|---|
| Execution control | Maintains target tension under high load | Often gets pulled into compensation by load |
| RPE judgment | Estimates reserve and fatigue accurately | Often misreads easy sets as effective or broken form as effort |
| Volume need | Few sets can be enough stimulus | Often needs more practice sets to find the target |
| Low-volume high-intensity | Can use it as a cycle phase | Should not copy it directly |
| Range choices | Makes range serve target tension | May assume bigger range or heavier load is always better |

Migration rule: when the user's base is insufficient, prioritize moderate volume + stable technique + controllable RPE rather than copying a high-level low-volume high-intensity template.

## Defaults when user data is missing

| Unknown field | Default handling |
|---|---|
| Training level unknown | Treat as ordinary trainee; do not give extreme low-volume high-intensity. |
| Recovery unknown | Start with medium volume and observe 2-3 weeks. |
| Technique unknown | Keep main lifts at or below RPE 8 and include technical cues. |
| Weak point unknown | Use balanced PPL without extra specialization. |
| Equipment unknown | Offer free-weight, machine, and cable substitutions. |

## Output requirements

When using this reference, include:

- Training-day structure: push/pull/legs or rolling order.
- Exercise selection: main lift, accessory, isolation, and weak-point slots.
- Sets and reps: never output exercise names only.
- RPE/RIR: separate main-lift and isolation effort.
- Rest: distinguish compounds and isolations.
- Progression: complete reps before adding load; use control, reps, density, or 10+5 for isolations.
- Technique cues: push day covers scapula and bench; pull day covers scapular depression and tension; legs day covers hip hinge and posterior chain.
- Adjustment rules: reduce volume, deload, or change emphasis when performance drops, joint pain rises, or soreness lasts too long.

## Reference signals

Use these only as supporting context; do not overstate them as medical certainty.

- ACSM resistance training guideline update: frequency, major muscle training, and sustainable planning.
- Split vs full-body training systematic review: when volume is equated, split structure alone is not automatically superior.
- Resistance-training frequency meta-analysis: frequency is mainly a way to distribute weekly volume.
- Training-to-failure reviews: failure is not always better and must be managed by movement type and fatigue.
- Technique-variable reviews: range of motion, tempo, and long-muscle-length loading matter, but must serve target tension.
- Rest-interval reviews: longer rest can preserve set quality, especially for demanding compound work.
