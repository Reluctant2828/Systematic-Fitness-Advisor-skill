# Body-Part Specialization Advanced Knowledge Base

Use this reference with `goal-specialization.md` when the user asks about 部位专攻, 弱项专项, 某块肌肉练不起来, 胸/肩/背/手臂/臀腿/小腿专项, weak point diagnosis, target-muscle feel, 4-8 week specialization blocks, or how to insert a priority muscle into an existing split.

## Core definition

部位专攻 is not "just add more sets to the weak part." It temporarily raises one target muscle's priority, frequency, and high-quality weekly volume for 4-8 weeks while reducing recovery cost from non-target muscles.

One-line rule: target muscle priority + non-target maintenance + recovery-resource redistribution.

| Dimension | General hypertrophy | Specialization block |
|---|---|---|
| Goal | Balanced whole-body progress | One or two weak points prioritized |
| Volume distribution | Relatively balanced | Target rises, non-targets maintain |
| Exercise choice | Basic coverage | Movements that create clear target tension |
| Order | Big muscles/main lifts first | Weak-point work can move to the start |
| Length | Long-term base plan | Usually 4-8 weeks, then reassess |
| Risk | Slower but stable | Too much volume or joint stress |

## Weak-point diagnosis

Do not assume a weak point only needs more work. Diagnose in this order:

1. Clarify the target: visual size, shape, strength, or carryover to a lift.
2. Check technique: is the target muscle being replaced by synergists or joints?
3. Check volume: are weekly hard sets enough and are they spread well?
4. Check recovery: sleep, nutrition, stress, soreness duration, and performance trend.
5. Then choose: add volume, add frequency, change exercises, lower load, or run a specialization block.

| Weak-point type | Signs | Direction |
|---|---|---|
| Low muscle mass | Size or visual fullness is clearly lagging | Raise weekly volume and frequency. |
| Poor recruitment | User cannot feel target; synergists dominate | Lower load, use stable machines/cables, unilateral work. |
| Technical weakness | Main movement path is unstable or joints compensate | Fix technique before adding volume. |
| Low frequency | Trained once weekly with too many sets in one day | Spread into 2-4 weekly exposures. |
| Recovery limitation | Target not recovered before the next session | Reduce total volume, improve sleep/food, or deload. |
| Joint limitation | Joint hurts before target fatigues | Substitute movement and avoid painful path. |

## Variables

| Variable | Range | Rule |
|---|---|---|
| Target weekly volume | Usually 12-24 hard sets/week | Add 2-4 sets/week from the current baseline, not all at once. |
| Non-target weekly volume | About 4-8 hard sets/week | Maintain without stealing recovery. |
| Target frequency | 2-4 exposures/week | Spread high-quality work instead of destroying the target in one session. |
| Compound intensity | RPE 7-9 / RIR 1-3 | Do not grind high-system-fatigue work to failure for long periods. |
| Isolation intensity | RPE 8-10 / RIR 0-2 | Drop sets, peak pauses, and lengthened partials can be used selectively. |
| Block length | 4-8 weeks | Reassess size, execution, recovery, and pain. |
| Deload | As fatigue requires | Reduce sets 30-50% and slightly lower intensity. |

Priority rules:

- Put weak-point work early in the session.
- Move non-priority work later or reduce it.
- If the target muscle cannot be felt, change execution or exercise before adding more sets.
- If joints complain, change the path or equipment before forcing progression.

## Chest specialization

Chest specialization is not just adding bench volume. Separate upper chest, overall thickness, stretch-position tension, and shoulder/triceps limitations.

| Problem | Signs | Strategy |
|---|---|---|
| Upper chest weak | Flat below clavicle; flat bench is okay but upper chest lacks fullness | Incline barbell/dumbbell, low-to-high cable fly, incline machine press. |
| Overall chest thin | Pressing strength is not terrible but chest size lags | Stable machine press + fly/pec deck + controlled eccentric. |
| Chest not felt | Front delts/triceps fatigue first | Lower load, add fly/pec deck pre-activation, improve scapular platform. |
| Bench stuck at bottom | Cannot press from chest | Pause bench, Spoto press, stretch-position chest strength. |
| Bench lockout stuck | Mid/late press cannot finish | Close-grip bench, dips, long-head triceps work. |

Sample chest/shoulder priority on PPL:

| Day | Focus | Logic |
|---|---|---|
| Push A | Bench technique + chest thickness | Bench/incline first, RPE around 8, chest 8-10 sets. |
| Pull | Back maintenance | Avoid excessive arm fatigue. |
| Legs | Leg maintenance | Reduce lower-body fatigue if it hurts push recovery. |
| Push B | Upper chest + stretch tension | Incline machine press, cable fly, controlled laterals. |
| Chest supplement | Low-fatigue pump | 6-8 sets of push-up/fly/pec deck, RIR 0-2. |

## Shoulder specialization

Shoulder specialization usually prioritizes middle and rear delts, not more heavy pressing. Pressing often covers front delts enough.

Rules:

- Lateral raises do not prioritize linear load progression.
- Progress by control, reps, pauses, eccentric, target tension, and reduced compensation.
- Avoid shrugging, swinging, or turning lateral raises into trap work.
- Use dumbbell lateral raise, cable lateral raise, and machine lateral raise for different resistance profiles.
- Middle delts often respond well to 3-5 small weekly doses better than one huge session.

| Movement | Role | Recommendation |
|---|---|---|
| Dumbbell lateral raise | Basic abduction | 12-25 reps, controlled eccentric. |
| Cable lateral raise | Sustained tension | High reps and peak pauses. |
| Machine lateral raise | Stable high tension | Drop sets and close-to-failure work. |
| Rear-delt fly | Rear delts | Stable scapula; do not turn it into a row. |
| Face pull | Rear delt/scapula | No shrugging; elbows open out. |

## Back specialization

First identify whether the weakness is lat width, mid-back thickness, rear delts, or scapular control.

| Goal | Priority movements | Cues |
|---|---|---|
| Lat width | Single-arm cable pulldown, pulldown, straight-arm pulldown | Scapular depression, elbow toward hip, sustained tension. |
| Mid-back thickness | Chest-supported row, seated row, barbell row | Scapular retraction, thoracic stability, elbow back. |
| Lower trap/scapular control | Face pull, Y raise, controlled pulldown | Depression, posterior tilt, no shrugging. |
| Rear delts | Reverse fly, rear-delt row | Upper arm abducts; lats do not steal the work. |
| Overall back strength | Pull-up, row, deadlift variant | No violent swinging; path stays stable. |

10+5 lat rule:

- Use on final set of single-arm pulldown or pulldown only when technique is stable.
- Do 10 standard reps, briefly reset, then add 5 quality reps.
- Progression order: improve 10+5 quality -> add more 10+5 sets -> small valid load increase.
- Priority: tension > scapular control > path > range > load.

Back specialization sample:

| Slot | Content |
|---|---|
| Pull A | Single-arm cable pulldown 4 x 10-15; pulldown 3 x 8-12; straight-arm pulldown 3 x 12-20; rear delts 3 x 15-25. |
| Push | Normal push work, avoid excessive shoulder/elbow fatigue. |
| Legs | Normal leg day, keep hard deadlift volume modest. |
| Pull B | Pull-up/pulldown 4 x 6-10; chest-supported row 3 x 8-12; final single-arm pulldown 10+5; biceps 4-6 sets. |
| Optional micro-dose | 2-3 sets straight-arm pulldown on non-pull day for lat skill practice. |

## Arms specialization

Arms specialization should distinguish biceps long head, short head, brachialis, triceps long head, lateral head, and elbow comfort.

| Target | Movements | Rule |
|---|---|---|
| Biceps overall | Barbell curl, dumbbell curl, preacher curl | Avoid shoulder swing; controlled eccentric. |
| Biceps long head | Incline dumbbell curl, behind-body cable curl | Use shoulder-extended position and lengthened tension. |
| Brachialis/forearm thickness | Hammer curl, reverse curl | Neutral grip; wrist stable. |
| Triceps long head | Overhead cable/dumbbell extension | Shoulder-flexed stretch. |
| Triceps lateral head | Rope pressdown, close-grip bench | Full elbow extension without shoulder shrugging. |

## Legs, glutes, calves specialization

Leg specialization must include hips, posterior chain, pelvic control, foot/ankle, and unilateral capacity, not only quads.

| Target | Priority movements | Notes |
|---|---|---|
| Quads | Front squat, hack squat, leg press, leg extension | Knee-dominant, controlled bottom tension. |
| Glute max | Hip thrust, Bulgarian split squat, squat variant, cable kickback | Hip extension without lumbar compensation. |
| Hamstrings | Romanian deadlift, seated/prone leg curl | Cover both hip extension and knee flexion. |
| Hip stability | Single-leg deadlift, Bulgarian split squat, lateral walk | Pelvic stability and knee tracking. |
| Calves | Standing calf raise, seated calf raise | Straight-knee for gastrocnemius, bent-knee for soleus. |

Rules:

- New trainees and fatigued users should not make conventional deadlift the only posterior-chain main lift.
- If hinge ability is poor, use RDL, single-leg deadlift, hip thrust, and back extension first.
- Include dynamic preparation for ankles, tibialis anterior, hips, pelvis, and core.
- Unilateral work develops pelvis, glute med, and foot/ankle control; it is not optional filler.

## Skills decision rules

| User situation | Judgment | Output |
|---|---|---|
| Target muscle under 8 sets/week | Likely volume too low | Raise to 10-14 sets/week first. |
| Target trained once/week with over 12 sets in one session | Low frequency and high session fatigue | Split into 2-3 exposures/week. |
| Target not felt and synergists fatigue first | Technique/exercise issue first | Use stable, unilateral, or lighter movements. |
| Joint hurts before muscle fatigues | Exercise mismatch or recovery issue | Substitute movement and reduce joint stress. |
| Performance drops for two weeks | Recovery insufficient or volume jumped too fast | Reduce 20-30% volume or deload. |
| Target recovers and performance improves | Continue progression | Add reps, then sets, then load. |

## Output requirements

When using this reference, include:

- Target muscle and weak-point type: size, recruitment, technique, frequency, recovery, or joint limitation.
- Block length, target weekly sets, maintenance sets, and frequency.
- Exercise slots and why they match the target.
- What volume is reduced elsewhere to free recovery.
- Progression order: execution -> reps -> sets/density -> load.
- Exit conditions: target improves, pain rises, recovery fails, or progress stalls.
