# Body Metrics Analysis

Use this reference when the user provides bodyweight, waist, measurements, progress photos, body composition estimates, sleep, steps, cardio, or asks whether fat loss, recomposition, body shaping, or a visual change is working.

## Purpose

Turn body metrics into trend decisions without overreacting to daily fluctuations.

## Metrics to extract

```text
date:
bodyweight:
7_day_average:
waist:
other_measurements:
photos:
body_fat_estimate:
steps:
cardio:
training_performance:
sleep:
hunger:
cycle/water/sodium notes:
```

## Reliability

| Metric | Reliability rule |
|---|---|
| 7-day average weight | More reliable than single weigh-ins. |
| Waist at same time/condition | Strong fat-loss signal when paired with photos. |
| Progress photos | Useful only with similar lighting, pose, distance, and timing. |
| Body-fat scale | Treat as rough trend, not exact truth. |
| Single-day weight spike | Usually water/glycogen/sodium/stool, not fat gain. |
| Training performance | Critical for muscle retention during cuts. |

## Trend windows

| Time window | Interpretation |
|---|---|
| 1-3 days | Noise only; do not adjust aggressively. |
| 1 week | Early signal; compare weekly averages. |
| 2-3 weeks | Enough for fat-loss plateau checks if adherence is known. |
| 4-8 weeks | Good window for recomposition and visual trend. |
| 8-12 weeks | Good point for full phase review or diet break/maintenance decision. |

## Fat-loss interpretation

| Pattern | Interpretation | Action |
|---|---|---|
| Weight average down, waist down, strength mostly stable | Good cut | Continue. |
| Weight down fast, strength crashing, hunger high | Deficit or fatigue too high | Raise calories slightly, reduce cardio/volume, improve recovery. |
| Weight stable, waist down, strength up | Recomposition likely | Continue if goal is shape, not scale speed. |
| Weight down, waist/photo unchanged | Could be water/glycogen or measurement error | Check adherence, protein, training, photos. |
| Weight unchanged 2-3 weeks, waist/photo unchanged, adherence good | Likely plateau | Follow plateau order from fat-loss reference. |
| Weight up after salty meal/leg day | Water retention likely | Wait and use weekly average. |

## Recomposition interpretation

Recomposition may be occurring when:

- Bodyweight is stable or slowly changing.
- Waist decreases or photos improve.
- Key lifts improve or stay stable.
- Measurements in target muscles improve.
- Adherence and protein are consistent.

Do not force aggressive scale loss if the user wants body shaping and recomposition signals are positive.

## Measurement standards

| Metric | Standard |
|---|---|
| Bodyweight | Morning, after bathroom, before food/drink when possible. |
| Waist | Same location, relaxed, same time, no intentional vacuum. |
| Photos | Front/side/back, same lighting, distance, pose, and time of day. |
| Limb/chest/hip measurements | Same tape tension and landmark. |
| Strength | Compare same exercise, similar technique, similar RPE/RIR. |

## Visual shaping checks

| Goal | Useful signals |
|---|---|
| Smaller waist | Waist trend, photos, bodyweight average, bloating notes. |
| Wider-looking shoulders | Side-delt/rear-delt training progression and photos. |
| Better V-taper | Waist down + lats/side delts maintained or growing. |
| Glute/leg shape | Hip/thigh measurements, posterior-chain training, photos. |
| Abs/lines | Waist, body fat trend, abdominal training consistency. |

## Adjustment logic

Fat loss:

1. Confirm adherence and tracking accuracy.
2. Check weekly average weight, waist, photos.
3. Check performance, hunger, sleep, stress.
4. If progress is good, do not change.
5. If plateau is real, increase steps/cardio before cutting more food.
6. If fatigue is high, reduce training/cardio stress or use maintenance/diet break.

Hypertrophy:

1. Check target muscle measurements/photos and training progression.
2. If bodyweight is not rising and strength/measurements stall, food may be too low.
3. If weight rises fast and waist jumps, surplus may be too high.
4. Use training-log analysis before changing bodyweight targets.

Powerlifting:

1. Bodyweight changes affect leverage and performance.
2. Cutting too fast can reduce SBD performance.
3. Near meet/test day, bodyweight manipulation should be conservative unless the user explicitly has a weight-class plan.

## Output requirements

When using this reference, include:

- Which metrics were available and their confidence.
- Trend conclusion: fat loss, recomposition, maintenance, likely water fluctuation, or plateau.
- What not to overreact to.
- The next adjustment, if any, and why.
- What to measure for the next 2-4 weeks.
