# Four-Day Split Advanced Knowledge Base

Use this reference with `goal-hypertrophy.md` and `hypertrophy-splits.md` when the user asks about 四分化, 4-day hypertrophy split, 上肢 A/下肢 A/上肢 B/下肢 B, upper/lower A/B, 推/拉/腿/弱项, 胸背腿肩手臂四分化, strength-biased plus hypertrophy-biased days, or how to design a 4-day split for natural trainees.

## Core definition

四分化 is not merely "training 4 days per week," nor is it automatically chest/back/legs/shoulders-arms. Advanced 四分化 uses four high-quality training units to distribute volume, frequency, intensity, movement patterns, weak-point work, and recovery.

Default recommendation: Upper A / Lower A / Upper B / Lower B. This is usually better for natural trainees than a body-part-only split because large muscles can receive about two weekly exposures while avoiding excessive low-quality volume in one session.

## Relationship to PPL

PPL organizes training into push, pull, and legs. 四分化 can split and reorganize the same logic to improve frequency and weak-point insertion.

| Dimension | PPL/三分化 | Advanced 四分化 |
|---|---|---|
| Base structure | Push / Pull / Legs | Upper A / Lower A / Upper B / Lower B, or Push / Pull / Legs / Weak Point |
| Advantage | Simple, clear, easy to run | Better volume distribution, higher per-session quality, easier weak-point insertion |
| Frequency | 3-6 sessions possible | Major muscles can more easily receive about 2 exposures/week |
| Risk | 3 days can be low frequency; 6 days can be high fatigue | Body-part version may become once-weekly frequency |
| Advanced execution | Main lifts keep reps in reserve; PPL can insert specialization | Strength day + hypertrophy day; compounds and isolations progress differently |

## Structure selection

| Structure | Best fit | Caution |
|---|---|---|
| Upper A / Lower A / Upper B / Lower B | Natural trainees, intermediate users, stable 4-day schedule | Default option when no strong reason says otherwise. |
| Push / Pull / Legs / Weak Point | User has a clear weak point and can recover | Weak-point day must have one priority; no random extra volume. |
| Chest / Back / Legs / Shoulders+Arms | Experienced body-part trainees who can execute high-quality sessions | Not default; may create once-weekly major-muscle frequency and junk volume. |

If the user prefers a traditional body-part split, modify it by adding small secondary stimulus on the shoulder/arms day, such as upper chest, back width, or rear delts, so key muscles get "main stimulus + secondary stimulus."

## Default week

| Day | Content | Positioning |
|---|---|---|
| Day 1 | Upper A | Strength-biased: bench, row, vertical pull, base push/pull capability |
| Day 2 | Lower A | Strength-biased: squat, hip hinge, posterior chain, core bracing |
| Day 3 | Rest or easy cardio/mobility | Recover joint and neural stress |
| Day 4 | Upper B | Hypertrophy-biased: chest/back second stimulus, side/rear delts, arm volume |
| Day 5 | Lower B | Hypertrophy-biased: quads, hamstrings, glutes, calves, unilateral stability |
| Day 6 | Rest or light weak-point supplement | Only when recovery supports it |
| Day 7 | Rest | Prepare next cycle |

## Training variables

Weekly volume is counted by muscle, not by training day.

| Muscle | Intermediate sets/week | Advanced sets/week | Notes |
|---|---|---|---|
| Chest | 10-14 | 14-20 | Count flat press, incline press, fly/pec deck together. |
| Back | 12-18 | 16-24 | Include vertical pulls and horizontal rows. |
| Quads | 10-16 | 14-22 | Count squat, leg press, leg extension. |
| Hamstrings | 8-14 | 12-18 | Need both hinge and leg curl patterns. |
| Middle delts | 8-16 | 12-24 | Tolerate high reps/frequency; no blind heavy loading. |
| Rear delts | 6-14 | 10-18 | Rear-delt fly, face pull, reverse pec deck. |
| Arms | 6-12 | 10-16 | Account for indirect push/pull work. |
| Calves | 8-16 | 12-24 | Standing and seated patterns can both be used. |

Intensity:

| Movement type | Recommendation |
|---|---|
| Bench, squat, deadlift variants | RPE 7-9 / RIR 1-3; do not grind to failure for long periods. |
| Row, pulldown, leg press, machine press | RPE 8-9 / RIR 1-2; target tension stays clear. |
| Lateral raise, fly, curl, pressdown | RPE 8-10 / RIR 0-2; final sets can be hard. |
| Drop sets, 10+5, rest-pause | Use mostly on final isolation sets, not high-risk compounds. |

Rest:

| Movement | Rest |
|---|---|
| Squat, bench, deadlift, heavy row | 2-4 min |
| Leg press, pulldown, machine press, seated row | 2-3 min |
| Lateral raise, curl, pressdown, fly | 60-120 sec |
| Supersets or pump work | 30-90 sec |

Progression: compounds prioritize load and reps; isolations prioritize control, reps, tension, and volume. Side raises, flyes, and rear-delt flyes should not progress like bench or squat.

## Upper A: strength-biased

Goal: develop bench, row, vertical pull, and upper-body stability while screening scapular position, thoracic extension, elbow/wrist path, and the full push-pull chain.

| Movement | Sets x reps | Intensity | Technique |
|---|---|---|---|
| Barbell bench press | 3-5 x 3-6 | RPE 7.5-8.5 | Scapula retracted/depressed, mild thoracic extension, leg drive, no excessive elbow flare. |
| Pull-up or pulldown | 3-4 x 6-10 | RPE 8 | Scapula depresses first; elbows pull down, no shrugging. |
| Barbell or chest-supported row | 3-4 x 6-10 | RPE 8 | Active back contraction; no lumbar swinging. |
| Incline dumbbell press | 2-3 x 8-12 | RPE 8-9 | Controlled eccentric, avoid anterior shoulder takeover. |
| Lateral raise | 3-4 x 12-20 | RPE 8-9 | Low compensation, stable path. |
| Triceps pressdown | 2-3 x 10-15 | RPE 8-9 | Elbow fixed and full extension. |
| Biceps curl | 2-3 x 10-15 | RPE 8-9 | No body swing; controlled eccentric. |

## Lower A: strength-biased

Goal: establish squat pattern, hip hinge, posterior-chain tension, and core bracing.

| Movement | Sets x reps | Intensity | Technique |
|---|---|---|---|
| Squat | 3-5 x 3-6 | RPE 7.5-8.5 | Knee/hip coordination, bracing, stable foot pressure. |
| Romanian deadlift | 3-4 x 6-10 | RPE 8 | Hips back, hamstrings lengthen, no low-back pulling. |
| Leg press | 3-4 x 8-12 | RPE 8-9 | Controlled bottom; quad bias. |
| Leg curl | 3 x 10-15 | RPE 8-9 | Hamstrings work; peak pause. |
| Standing calf raise | 4 x 8-15 | RPE 8-9 | Bottom stretch and top contraction. |
| Anti-extension/anti-rotation core | 2-4 sets | Technical | Plank, dead bug, Pallof press. |

## Upper B: hypertrophy-biased

Goal: add chest/back second stimulus, side/rear delt work, and arm volume with clear target tension, stretch, peak contraction, and local fatigue.

| Movement | Sets x reps | Intensity | Technique |
|---|---|---|---|
| Incline press or machine chest press | 3-4 x 8-12 | RPE 8-9 | Upper-chest tension; shoulder does not roll forward. |
| Pulldown | 3-4 x 10-15 | RPE 8-9 | Scapular depression and sustained lat tension. |
| Seated row | 3-4 x 10-15 | RPE 8-9 | Elbows move back; do not just pull the handle. |
| Cable fly | 2-4 x 12-20 | RPE 9 | Stretch control and adduction. |
| Lateral raise | 4-6 x 15-30 | RPE 9-10 | High reps, low compensation; drop set optional. |
| Rear-delt fly | 3-5 x 12-25 | RPE 9 | Avoid trap dominance. |
| Overhead triceps extension | 2-4 x 10-20 | RPE 9 | Long-head triceps stretch. |
| Incline curl | 2-4 x 10-20 | RPE 9 | Biceps long-head stretch. |

## Lower B: hypertrophy-biased

Goal: reduce neural pressure while increasing local stimulus for quads, hamstrings, glutes, calves, and unilateral stability.

| Movement | Sets x reps | Intensity | Technique |
|---|---|---|---|
| Hack squat or front squat | 3-4 x 6-10 | RPE 8-9 | Quad bias and stable trunk. |
| Bulgarian split squat | 2-4 x 8-12 | RPE 8-9 | Pelvis stable; knee/hip path controlled. |
| Leg extension | 3-5 x 12-20 | RPE 9-10 | Peak contraction and controlled eccentric. |
| Seated leg curl | 3-5 x 10-20 | RPE 9 | Lengthened hamstring position; hips stay down. |
| Hip thrust or glute bridge | 2-4 x 8-15 | RPE 8-9 | Peak glute contraction, no lumbar compensation. |
| Seated calf raise | 4-6 x 12-25 | RPE 9 | Soleus volume; slow eccentric. |
| Crunch or hanging leg raise | 2-4 sets | Technical | Control posterior pelvic tilt. |

## Weak-point insertion

| Weak point | Fourth-day strategy | Control rule |
|---|---|---|
| Shoulders | 6-10 sets side delts + 4-6 sets rear delts + 2-3 light press sets | Side/rear delts progress by control and reps, not blind load. |
| Chest | 3-4 sets incline press + 3-5 sets fly/pec deck + 2-3 machine/press-up sets | Avoid conflict with heavy push day. |
| Lat width | 6-10 sets vertical pull + 3-5 sets straight-arm pulldown | Scapular depression and lat tension first. |
| Arms | 6-8 sets biceps + 6-8 sets triceps | Vary elbow/shoulder positions for long head, short head, and brachialis. |

## Periodization and deload

| Stage | Execution |
|---|---|
| Week 1 | Start near minimum effective volume; compounds RIR 2-3. |
| Week 2 | Add 1-2 hard sets to priority muscles or add reps at the same load. |
| Week 3 | Main hypertrophy zone; compounds RIR 1-2, isolations can be RIR 0-1. |
| Week 4 | Higher volume; safe isolations can use 10+5 or drop sets. |
| Week 5 | Decide whether to peak volume; if performance drops, stop adding. |
| Week 6 | Deload: volume down 40-60%, intensity held or slightly reduced. |

Deload signals:

- Same load and movement performance drops for 2-3 consecutive sessions.
- Warm-up weights feel unusually heavy.
- Joint discomfort increases or soreness lasts beyond 72 hours.
- Sleep, appetite, and training drive fall.
- Target-muscle pump drops while whole-body fatigue rises.

## Common mistakes

| Mistake | Correction |
|---|---|
| Turning 四分化 into once-weekly body-part training | Use upper/lower A/B or add secondary stimulus. |
| Too many exercises and too few trackable sets | Keep core movements and track load, reps, RIR, execution. |
| Every movement to failure | Compounds keep 1-3 RIR; isolations can go closer. |
| Copying advanced low-volume high-intensity work | Ordinary trainees need enough sets to learn target tension. |
| Back work only chases load | Scapular control, path, and sustained tension first. |
| Leg work is only squat/leg press | Add hinge, leg curl, unilateral, calves, and core. |
| Isolation load jumps too fast | Use reps, eccentric, peak pauses, and volume instead. |

## Output requirements

When using this reference, include:

- Chosen 四分化 structure and why it fits: upper/lower A/B, PPL+weak point, or modified body-part split.
- Weekly schedule and recovery spacing.
- Exercise slots with sets, reps, RIR, rest, and progression.
- Which days are strength-biased versus hypertrophy-biased.
- Weak-point insertion only when it has one clear priority and recovery room.
