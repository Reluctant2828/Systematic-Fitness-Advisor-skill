# Hypertrophy Split Selector

Use this reference with `goal-hypertrophy.md` when the user asks how to arrange 增肌 split structure, such as 二分化, 三分化, 四分化, 五分化, PPL, upper/lower, or weekly training days.

## Selection inputs

Prioritize:

- Weekly training days and session length.
- Training age and technical consistency.
- Recovery: sleep, soreness, joint stress, stress, skipped sessions.
- Target muscles and weak points.
- Current weekly sets and frequency per muscle.
- Equipment availability and crowded-gym constraints.

## Core rule

Choose the split that lets the user hit recoverable weekly volume with high-quality sets and repeatable adherence. Do not choose a split because it sounds advanced.

## 二分化

Common forms:

- Upper/lower: 上肢 / 下肢.
- Push/pull: 推 / 拉, with legs placed according to recovery.

For detailed Upper/Lower execution, read `split-two-division.md` when the user asks about 上肢 A/B, 下肢 A/B, horizontal push/pull, knee-dominant, vertical push/pull, hip-dominant, 3-day rolling 二分化, 4-day standard 二分化, or weak-point insertion.

Best fit:

- 3-4 training days per week.
- User wants each muscle about 2 times per week.
- Recovery or schedule is inconsistent.
- Beginner to intermediate users, or advanced users returning from a break.

Strengths:

- High practice frequency.
- Easy to adjust volume.
- Good for balanced hypertrophy and recomposition.

Risks:

- Sessions can become too long if every muscle is overloaded.
- Lower body may be underplanned if using push/pull without clear leg placement.

Default weekly examples:

- 4 days: 上肢 / 下肢 / 上肢 / 下肢.
- 3 days: 上肢 / 下肢 / 上肢, then next week 下肢 / 上肢 / 下肢.
- Advanced 4 days: 上肢 A / 下肢 A / 上肢 B / 下肢 B.
- 5 days: standard 4-day Upper/Lower plus one weak-point day only when recovery supports it.

## 三分化

Common forms:

- PPL: push / pull / legs.
- Chest-back / legs / shoulders-arms.
- Target-biased 3-way split when one body part needs more work.

For practical PPL execution, read `ppl-practical.md` when the user asks about push day, pull day, legs day, bench press, lat tension, scapular control, hip/posterior-chain leg training, 10+5 progression, rolling PPL, specialization insertion, or ordinary-vs-advanced trainee adaptation.

Best fit:

- 3-6 training days per week.
- User wants more per-session focus than 二分化.
- User can recover from moderate muscle-specific volume.

Strengths:

- Easy to organize movement patterns.
- PPL can run once weekly for 3 days or twice across 6 days.
- Good balance between frequency and focused volume.

Risks:

- 3 days/week PPL hits each muscle only once if not modified.
- 6 days/week PPL can accumulate fatigue quickly.

Default weekly examples:

- 3 days: push / pull / legs.
- 5 days rolling: push / pull / legs / rest / push / pull, then continue legs next week.
- 6 days: push / pull / legs / push / pull / legs.
- Advanced 6-day: push A / pull A / legs A / rest / push B / pull B / legs B.
- Conservative 3-day: push / pull / legs with technique and habit-building emphasis.

Execution defaults:

- Push day = chest + middle delts + triceps, with bench or another stable press as the usual main lift.
- Pull day = back + rear delts + biceps, with lat tension and scapular control prioritized over load.
- Legs day = lower body + hip/posterior chain + stability, with unilateral and hinge capacity built before chasing heavy deadlifts.
- PPL can accept a periodic chest or shoulder specialization day when weak points and recovery justify it.

## 四分化

Common forms:

- Upper/lower repeated with different emphases.
- Chest+triceps / back+biceps / legs / shoulders+arms.
- Push / pull / legs / weak-point day.

For detailed 四分化 execution, read `split-four-division.md` when the user asks about Upper A / Lower A / Upper B / Lower B, strength-biased plus hypertrophy-biased days, PPL + weak point, or whether traditional chest/back/legs/shoulders-arms is appropriate.

Best fit:

- 4 training days per week.
- User wants one or two priority muscles without going to 5+ days.
- Intermediate users who need more focus than 二分化 but still need recovery days.

Strengths:

- Easier to give weak points a dedicated slot.
- Good compromise between frequency, focus, and recovery.
- Works well for busy users with fixed 4-day schedules.

Risks:

- If designed like a bro split, some muscles may get too little frequency.
- Arm/shoulder volume can overlap heavily with chest/back.

Default weekly examples:

- Upper strength-biased / lower / upper hypertrophy-biased / lower+weak point.
- Chest+triceps / back+biceps / legs / shoulders+arms.
- Push / pull / legs / target muscle.
- Preferred natural-trainee default: 上肢 A / 下肢 A / 上肢 B / 下肢 B.

## 五分化

Common forms:

- Chest / back / legs / shoulders / arms.
- Chest+front delt / back+rear delt / legs / shoulders / arms.
- Glutes or another priority body part gets its own day.

For detailed 五分化 execution, read `split-five-division.md` when the user asks about chest day, back day, legs day, shoulder day, arms/weak-point day, secondary stimulus, Day 5 placement, or whether the user is ready for five training days.

Best fit:

- 5 training days per week.
- User has stable sleep, nutrition, and schedule.
- User needs high focus per session or body-part specialization.
- Intermediate to advanced users who can judge target-muscle execution.

Strengths:

- High local focus and exercise variety.
- Easy to prioritize weak parts without exhausting the whole body each session.
- Good for users who enjoy shorter, focused sessions.

Risks:

- Many muscles may only be trained once per week unless overlap is planned.
- Missing one session can leave a body part undertrained for the week.
- Not automatically better than 二分化 or 三分化.

Default weekly example:

- Chest / back / legs / shoulders / arms, with weak-part mini-volume placed on a second day when recovery allows.
- Advanced default: chest + light triceps / back + light biceps / legs A / shoulders + rear delts / arms + weak point or legs B.

## Decision guide

- 2-3 days/week: use full body or 二分化 rotation.
- 4 days/week: use 二分化 upper/lower or 四分化.
- 5 days/week: use 三分化 rolling, 四分化+weak point, or 五分化.
- 6 days/week: use PPL twice or a specialization split, but monitor fatigue closely.

If the user asks "which split is best", answer with the split that best matches their schedule and recovery, then give one reason against the other likely options.

## Output requirements

When recommending a split, include:

- Split name and why it fits.
- Weekly schedule.
- Muscle frequency.
- Priority muscles and weekly hard-set targets.
- Main exercise slots, not necessarily every exercise if data is sparse.
- When to switch split: adherence failure, recovery issue, plateau, or goal change.
