# Goal Module: Body-Part Specialization

Use this module when the user wants to prioritize one body part, improve a lagging muscle, or run a focused block for chest, back, shoulders, glutes, arms, legs, abs, or another area. Read `specialization-advanced.md` when the request asks for weak-point diagnosis, target-muscle feel, chest/shoulder/back/arms/glutes/legs/calf specialization, or how to insert a priority muscle into an existing split.

## Primary objective

Move recovery resources toward one or two priority muscles for 4-8 weeks while maintaining the rest of the body.

## Inputs to prioritize

- Priority body part and reason: size, shape, strength, symmetry, or weak-point carryover.
- Current weekly sets for the target and non-target muscles.
- Exercise list that the user feels in the target muscle.
- Recovery cost, joint tolerance, and schedule.
- Recent photos, measurements, or performance trend if available.

Read `specialization-advanced.md` when the request involves:

- 部位专攻, 弱项专项, 某块肌肉练不起来.
- Chest, shoulder, lat/back, arms, glutes, quads, hamstrings, calves specialization.
- Target muscle has no feeling, synergists take over, or joints hurt first.
- 4-8 week priority blocks, target weekly sets, maintenance volume, or split insertion.
- Combining SBD strength work with a physique weak point.

## Programming defaults

- Priority muscle volume: often 12-22 hard sets per week if recovery supports it.
- Maintenance muscles: often 4-8 hard sets per week.
- Frequency: 2-4 target exposures per week.
- Exercise mix: one high-tension slot, one stable volume slot, one isolation/pump or lengthened-position slot when appropriate.
- Block length: 4-8 weeks, then reassess.

## Progression

- Improve target-muscle execution first.
- Then progress reps, load, density, pauses, range of motion, or set count.
- Avoid chasing load if it shifts tension away from the target muscle.

## Adjustment rules

- Reduce non-priority volume before adding more total weekly training.
- Stop adding target sets if soreness lingers, joints complain, or target performance drops.
- Rotate exercises by movement pattern and equipment when the target stimulus fades.
- Exit the block when the target improves, recovery cost is too high, or progress stalls after reasonable adjustments.

## Output emphasis

Report priority muscle, block length, target weekly sets, maintenance sets, exercise slots, progression rule, and exit conditions.
