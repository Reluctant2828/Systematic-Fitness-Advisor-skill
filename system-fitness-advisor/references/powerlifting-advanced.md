# Powerlifting Advanced Knowledge Base

Use this reference with `goal-powerlifting.md` when the user asks about 力量举, SBD, 深蹲卧推硬拉, 1RM, e1RM, peaking, meet prep, top single, back-off sets, DUP, attempts, squat/bench/deadlift technique, or how to combine powerlifting with hypertrophy or body-part specialization.

## Core definition

Powerlifting targets the competition-style 1RM total in squat, bench press, and deadlift. It differs from bodybuilding because the core goal is not pump or target-muscle feel; it is lifting the most weight with rule-compliant technique.

Core model:

- Competition specificity.
- Repeated technical practice.
- Planned high-intensity exposure.
- Fatigue management.
- Peaking/tapering so strength appears on test or meet day.

Rules:

- Squat, bench, and deadlift or close variations must remain in the program unless pain/equipment prevents them.
- Training is not weekly max testing. Use RPE, e1RM, top singles, and back-off sets.
- Bench often tolerates higher frequency than squat/deadlift.
- Deadlift has high fatigue cost; hard pulls must be dosed carefully.
- Accessories serve SBD weak points, muscle mass, technical stability, and injury resilience.

## Powerlifting vs hypertrophy/specialization

| Dimension | Hypertrophy/specialization | Powerlifting |
|---|---|---|
| Goal | Muscle size, shape, weak points | Squat/bench/deadlift 1RM and total |
| Exercise choice | Target-muscle tension first | Competition specificity first |
| Feedback | Pump, circumference, target fatigue | e1RM, RPE, bar speed, technical standards |
| Reps | More moderate/high reps | Many 1-6 rep sets |
| Failure | Isolation can approach failure | Main lifts avoid frequent failure |
| Cycle | Volume and hypertrophy | Technique, strength, peaking, meet prep |

## Technique: squat

Squat goal: reach rule-compliant depth and stand up with stable bar path and full-body tension.

| Cue | Requirement |
|---|---|
| Stance | Adjust by hip structure and style; stable descent and force output. |
| Bar position | High bar is more upright/quad-biased; low bar often suits maximal powerlifting loads. |
| Brace | Build abdominal pressure and back tension before descent; do not leak tension in the hole. |
| Descent | Hips and knees coordinate; no uncontrolled lumbar collapse or knee cave. |
| Depth | Hip crease below top of knee for competition standard. |
| Ascent | Chest/back tension maintained; hips and shoulders rise together. |

Squat weak points:

| Sticking point | Possible cause | Fixes |
|---|---|---|
| Stuck in the bottom | Poor bottom tension, weak quads/hips | Pause squat, front squat, hack squat, leg press. |
| Folding mid-range | Trunk/back support insufficient | Safety bar squat, good morning, anti-flexion core. |
| Knee cave | Glute med/foot-ankle control issue | Bulgarian split squat, lateral walk, tempo squat. |
| Unstable depth | Hip/ankle mobility or control issue | Pause squat, dynamic warm-up, ankle dorsiflexion. |
| Too much forward lean under load | Center of mass or upper-back tension loss | High bar/front squat, chest-supported row, upper-back work. |

## Technique: bench press

Bench is a full-body powerlifting movement: scapular platform, thoracic arch, leg drive, wrist/elbow path, and pause control.

| Cue | Requirement |
|---|---|
| Unrack | Keep scapular position; do not lose shoulder platform pulling the bar out. |
| Scapula | Retracted, depressed, stable on bench. |
| Arch | Use thoracic extension to reduce range and optimize force; do not force lumbar-only extension. |
| Leg drive | Feet stable; build full-body tension. |
| Grip | Choose by shoulder/elbow comfort and rules; do not blindly go max-wide. |
| Descent | Controlled to chest; keep tension for pause command. |
| Press | Press upward/back while triceps finish lockout. |

Bench weak points:

| Sticking point | Possible cause | Fixes |
|---|---|---|
| Off chest | Weak pause/bottom tension | Pause bench, Spoto press, long-pause bench. |
| Mid-range | Weak chest/shoulder/triceps transition | Close-grip bench, dumbbell press, incline press. |
| Lockout | Triceps strength | Close-grip bench, dips, pressdown, overhead extension. |
| Shoulder pain | Scapula unstable or grip mismatch | Adjust grip, add rear delt/scapula work, reduce risky movements. |
| Path breaks down when heavy | Technical consistency issue | Lighter pause work and speed/technique sets. |

## Technique: deadlift

Deadlift goal: start from the floor with maximum load while keeping back tension, hip/knee position, close bar path, wedge, and controlled lockout.

| Cue | Requirement |
|---|---|
| Stance | Conventional or sumo based on structure and leverage. |
| Bar position | Near mid-foot; bar stays close after the start. |
| Pull slack | Remove slack before the pull; build lat and trunk tension. |
| Hip/shoulder position | Hips neither too low nor too high; shoulders fit the chosen style. |
| Start | Push the floor away rather than jerking the bar. |
| Lockout | Finish hip extension; do not over-lean back. |

Deadlift weak points:

| Sticking point | Possible cause | Fixes |
|---|---|---|
| Off floor | Start position and leg drive issue | Pause deadlift, deficit deadlift, leg press/quad work. |
| Around knees | Bar drifts or lats loosen | Pause deadlift, RDL, rows, straight-arm pulldown. |
| Lockout | Glutes/hamstrings or upper-back tension | Rack pull, hip thrust, RDL, back extension. |
| Low-back fatigue | Hinge/bracing issue or too much hard pulling | Reduce deadlift frequency, add core and technique work. |
| Sumo hip access issue | Hip stability/adductor limitation | Sumo pause pulls, hip mobility, adductor/glute med work. |

## Training variables

| Intensity | Use | Typical arrangement |
|---|---|---|
| 50-65% 1RM | Technique, speed, recovery | Speed work, technique practice, deload. |
| 65-75% | Volume accumulation | 5-8 rep sets with quality. |
| 75-85% | Strength base | 3-6 rep main work. |
| 85-92% | High-intensity adaptation | 1-3 reps, RPE controlled. |
| 92%+ | Peak/test | Few singles; not long-term volume. |

| Training type | Suggested RPE | Meaning |
|---|---|---|
| Technical volume | RPE 6-7 | Practice and volume without heavy fatigue. |
| Main work | RPE 7-8.5 | Most training lives here. |
| High-intensity single | RPE 7-9 | Maintain heavy skill and estimate e1RM. |
| Test/meet | RPE 9.5-10 | Only for testing or competition. |
| Accessories | RPE 7-10 | Depends on risk and recovery. |

Top single + back-off:

- Use top single @RPE 7-8 to practice heavy skill and estimate daily readiness.
- Then perform back-off sets for volume and technique.
- If top single is slower or higher RPE than expected, reduce back-off load or volume.
- Do not chase e1RM if depth, pause, lockout, or bar path is not credible.

## Periodization

| Phase | Length | Goal | Training features |
|---|---|---|---|
| Base/volume | 4-8 weeks | Muscle, technical volume, work capacity | Medium intensity, more 4-8 rep sets, more variations. |
| Strength | 4-6 weeks | Specific strength | More competition lifts, more 3-5 rep work. |
| Peak | 2-4 weeks | High-intensity singles | More singles/doubles, fewer accessories. |
| Taper/test | 1-2 weeks | Drop fatigue, maintain neural readiness | Lower volume, some high-intensity exposure. |
| Recovery | 1-2 weeks | Joints and psychology | Low-intensity technique and general work. |

DUP:

- Daily undulating periodization uses different goals inside the week, such as volume day, intensity day, and technique day.
- It suits powerlifting because SBD need frequent practice, but not every day can be maximal.

Sample four-day week:

| Day | Squat | Bench | Deadlift |
|---|---|---|---|
| Day 1 volume | 4 x 6 @70-75% | 5 x 5 @70-75% | - |
| Day 2 deadlift | Light technique 3 x 5 | Pause bench 4 x 4 | 4 x 4 @72-80% |
| Day 3 intensity | Top single @8 + 3 x 3 | Top single @8 + 4 x 3 | - |
| Day 4 variations | Pause squat 3 x 4 | Close-grip bench 4 x 6 | RDL 3 x 6-8 |

Alternative four-day logic:

| Day | Main lift | Accessories |
|---|---|---|
| Day 1 | Squat main + bench volume | Quads, core, upper back |
| Day 2 | Deadlift main + bench variation | Hamstrings, lats, biceps |
| Day 3 | Bench main + squat variation | Triceps, rear delts, chest/back balance |
| Day 4 | Squat/deadlift weak-point variation | Posterior chain, upper back, scapular control |

## Twelve-week outline

| Weeks | Goal | Execution |
|---|---|---|
| 1-4 | Volume and technique | 65-78%, 4-8 reps, stable standards. |
| 5-8 | Strength | 75-88%, 2-5 reps, top singles @7-8. |
| 9-10 | Peak prep | 85-92%, more singles/doubles, fewer accessories. |
| 11 | Taper | Volume down 40-60%, keep a little heavy exposure. |
| 12 | Test or meet | Use attempt logic. |

## Attempt selection

| Attempt | Goal | Selection |
|---|---|---|
| Opener | Secure success | About 90-92% target 1RM; a weight the user reliably completes in training. |
| Second | Near current best | About 96-100% current ability, adjusted by opener speed. |
| Third | PR attempt | Based on second-attempt speed and day readiness; do not add blindly. |

## Accessories

Accessories are not "more exercises to get tired." They serve weak points, muscle mass, technical stability, and injury resilience.

| Main lift | Accessories | Purpose |
|---|---|---|
| Squat | Pause squat, front squat, leg press, Bulgarian, anti-flexion core | Bottom strength, quads, trunk stability. |
| Bench | Pause bench, close-grip bench, Spoto press, rows, triceps pressdown | Bottom control, triceps lockout, scapular stability. |
| Deadlift | Pause deadlift, deficit deadlift, RDL, hip thrust, row | Start, lats, posterior chain, lockout. |
| General | Rear delts, face pull, bracing, back extension, hip stability | Injury resilience and technical stability. |

## Common errors

- Treating every week as 1RM testing.
- Only training the three main lifts and ignoring weak-point assistance.
- Adding too many accessories and stealing main-lift recovery.
- Hard deadlifting too frequently and keeping the low back fatigued.
- Treating bench as chest-only and ignoring scapula, leg drive, pause, and triceps lockout.
- Squatting by load only, ignoring depth, bracing, and bottom stability.
- Ignoring meet rules: bench pause, squat depth, deadlift lockout and down command.
- Keeping high-volume accessory work too close to meet/test day.

## Mixed powerlifting + specialization

Use when the user wants SBD strength and better body parts.

| Goal combo | Main logic | Accessory logic |
|---|---|---|
| Bench stronger + bigger chest | Bench is the main lift with controlled RPE | Incline press and fly/pec deck for chest volume. |
| Squat stronger + bigger legs | High-specificity squat work | Leg press, hack squat, leg extension for quads. |
| Deadlift stronger + posterior chain | Low-to-moderate frequency high-quality deadlift | RDL, hip thrust, leg curl for posterior chain. |
| Total strength + shoulder width | SBD first | Small frequent lateral/rear-delt work. |
| Meet prep + maintain physique | Peak SBD first | Accessories reduced to minimum effective dose. |

Mixed four-day template:

| Day | Main lift | Specialization |
|---|---|---|
| Day 1 | Squat main + bench volume | Quads/chest volume. |
| Day 2 | Deadlift main | Lats, hamstrings, biceps. |
| Day 3 | Bench main + squat variation | Triceps, middle delts, rear delts. |
| Day 4 | Squat/deadlift weak-point variation | Target weak point 6-10 sets. |

Adjustment rules:

- If SBD performance drops while weak-point volume is high, cut specialization volume first.
- If target muscle does not improve because main lifts dominate recovery, reduce back-off volume.
- If joint pain appears, substitute accessories and reduce main-lift intensity.
- If training days are fewer than 4, use full-body or upper/lower for strength before PPL.
- If training days are 5-6 and hypertrophy is primary, use PPL with inserted SBD technique or weak-point days.
